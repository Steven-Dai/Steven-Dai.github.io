msyql登录服务器重置root登录密码

# 打开配置文件
```
vi /etc/my.cnf
```

# 添加一行配置
```
[mysqld]
skip-grant-tables
```

# 重启mysql
```
systemctl restart mysqld
```

# 修改密码
```
mysql
mysql> use mysql;
mysql> update user set password=password("123") where user="root";
mysql> flush privileges;
```

# 重启服务
```
删除skip-grant-tables配置
systemctl restart mysqld
```
