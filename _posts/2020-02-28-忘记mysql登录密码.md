title: 忘记mysql的root密码
---

1、打开配置文件

```
vi /etc/my.cnf
```

2、添加一行配置

```
[mysqld]
skip-grant-tables
```

3、重启mysql

```
systemctl restart mysqld
```

4、修改密码

```
mysql
mysql> use mysql;
mysql> update user set password=password("123") where user="root";
mysql> flush privileges;
```

5、重启服务

```
删除skip-grant-tables配置
systemctl restart mysqld
```
